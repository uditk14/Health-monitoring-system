Welcome to E-Health Care System

E-Health Care is a modern project that assigns qualified and most appropriate doctors to the patients. It uses a device that is inbuilt with sensors. It senses patient's pulse rate and transmits it to the main database via Bluetooth connectivity. This project has been made availabe in both application and website formats.

Patients and Doctors need to sign up on either website or on our application. The patient is able to feed data manually and is also able to synchronise with the our device. The doctors are able to check the heart rate report of the patient. After successful undergo system automatically assigns a doctor to the patient.
